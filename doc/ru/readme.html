Idea word Separators

Это дополнение исправляет проблему с навигацией по словам в IntelliJ IDEA и Android studio, т.е. NVDA перестанет многократно повторять слова, которые прилегают к различным знакам пунктуации. Такое поведение достигается путем установки для NVDA таких же разделителей слов как и в IDEA.

Настройка IDEA

Дополнение будет корректно функционировать только, если навигация по словам работает в windows стиле, т.е. при перемещении к следующему или предыдущему слову курсор устанавливается на его первую букву . в IntelliJ IDEA и Android studio по умолчанию используется другой режим, но эти настройки можно изменить следующим образом:

  1. открыть "settings" из меню файл или нажав сочитание клавиш "ctrl+alt+s"
  2. Перейти по пути editor/general
  3. в списке "When movings by word" выбрать "always jump the word start windows defoult".
